# Networking práctico — Tailscale, NetBird y ZeroTier

## Resumen
Comparativa práctica y casos de uso con ejemplos de configuración y troubleshooting.

## Casos cubiertos
- Conexión punto a punto para administración remota.
- Redes superpuestas para microservicios en entornos heterogéneos.

## Ejemplo rápido (Tailscale)

```bash
# Instalar tailscale
dpkg -i tailscale_latest_amd64.deb
tailscale up
```
