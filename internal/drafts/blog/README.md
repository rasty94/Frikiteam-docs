# Blog Drafts

Este directorio contiene borradores de posts para el blog de WordPress (https://frikiteam.es).

## Uso

1. Escribe y revisa posts técnicos aquí en formato Markdown
2. Una vez listos, publícalos manualmente en WordPress
3. Mantén los archivos aquí como backup/referencia

## Estructura

- `2025/` - Posts del año 2025
- Más carpetas por año según sea necesario
